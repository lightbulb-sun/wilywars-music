# Wily Wars (Mega Drive) Music Hack

## Changes
Restores the unused music track 72
for the select your weapon Light Lab screen in Wily Tower
in the European version of the Sega Mega Drive game
*Mega Man: The Wily Wars*.

## Download
The latest release can be found on the
[releases page](https://github.com/lightbulb-sun/wilywars-music/releases).
Patch the original ROM with one of the `.bsdiff`, `.ips` or `.bps` files
to create the hack.
